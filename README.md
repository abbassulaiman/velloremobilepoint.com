# Vellore Mobile Point Website

## Deployment Instructions

### Steps to Deploy Changes

1. Open the hosting control panel using the provided URL, username, and password.

2. Navigate to **File Manager**.

3. Open the following folder:

   ```
   velloremobilepoint.com
   ```

4. Locate the file(s) that were modified in the latest update.

5. Upload and replace the existing file(s) with the updated version(s).

6. Confirm that the upload has completed successfully.

7. Clear any website or server cache if applicable.

8. Visit the website and verify that the changes are reflected correctly.

---

## Hosting Access

The hosting URL, username, and password will be shared separately through a secure channel.
